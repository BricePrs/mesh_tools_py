# mesh_tools_py

## Requirement
 + Python3
 + pip3
 + maturin
 + rustc
 + rustup?
 + sdl2

## How to run

The first you need to run first
```bash
python3 -m venv .env
```


For each terminal run
```bash
source .env/bin/activate
```

To compile/run
```bash
maturin develop
python3 examples/example.py
```
